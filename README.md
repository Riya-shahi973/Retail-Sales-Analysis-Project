# Retail Sales Analysis Project

## Project Overview

This is an end-to-end Data Analytics project built using Excel, SQL, Python, and Power BI. The project analyzes retail sales data and provides business insights through an interactive dashboard.

## Tools Used

* Excel
* SQL
* Python (Pandas, NumPy, Matplotlib)
* Power BI
* DAX

## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. SQL Analysis
5. Dashboard Creation in Power BI

## Key Features

* KPI Cards (Sales, Profit, Orders, Customers)
* Sales Trend Analysis
* Customer Analysis Dashboard
* Region-wise Drill-through Dashboard
* Interactive Filters and Visualizations

## Files Included

* Excel files
* SQL queries
* Python notebooks/scripts
* Power BI dashboard (.pbix)
* Dataset files
